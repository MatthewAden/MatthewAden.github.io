vscode远程推送文件至github

发表于2022-01-03|更新于2022-01-18|编程中遇到的问题
|字数总计:258|阅读时长:1分钟

1.在github上新建一个仓库

2.在本地用编辑器打开本地的文件夹

3.此时如需把文件夹中的文件推送至远程仓库，

git init                      #初始化一个本地仓库
git add readme.txt            #将这一个文件上传至仓库
#git add .                    #将所有文件上传至仓库
git commit -m “first commit”  #first commit是备注 即第一次提交
git branch -M main
git remote add origin https://github.com/xxx/xxx.git  #加仓库名
git push -u origin main

4.此时如果修改了文件 vscode左侧源代码管理会出现蓝色圆圈及其数字 表示文件已经更改，我们就需要将这个改变的文件先暂存本地仓库(源代码管理处文件的加号)再提交到本地仓库(此时会弹出输入框，输入备注回车) ，最后点击最左下角状态栏的循环箭头即可提交github更新更改

注:如果非第一次创建只需执行第四步

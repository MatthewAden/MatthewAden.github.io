drop函数基本介绍

  • 用于删除数据集中多余的数据
DataFrame.drop(labels=None, axis=0, index=None, columns=None, inplace=False)

labels:待删除的行名or列名；
axis:删除时所参考的轴，0为行，1为列；
index:待删除的行名
columns:待删除的列名
inplace:布尔值，默认为False,这是返回的是一个copy;若为True,返回的是删除相应数据后的版本

lambda函数

  • lambda 函数是一种小的匿名函数，lambda 函数可接受任意数量的参数，但只能有一个表达式。
#例1
# 语法lambda arguments : expression
x = lambda a, b, c : a + b + c
print(x(5, 6, 2))

#例2
def myfunc(n):
  return lambda a : a * n
mydoubler = myfunc(2)
print(mydoubler(11))

map函数

  • map是python内置函数，会根据提供的函数对指定的序列做映射，map()函数的格式是
map(function,iterable,...)
    def square(x) :                            #计算平方数
        return x ** 2
map(square, [1,2,3,4,5])                       #计算列表各个元素的平方
<map object at 0x100d3d550>                    #返回迭代器
list(map(square, [1,2,3,4,5]))                 #使用list() 转换为列表
>>>[1, 4, 9, 16, 25]
list(map(lambda x: x ** 2, [1, 2, 3, 4, 5]))   # 使用 lambda 匿名函数
>>>[1, 4, 9, 16, 25]

#map()还能进行类型转换
list(map(int, (1, 2, 3)))
>>>[1, 2, 3]
#可以提取字典中的key
list(map(int,{'1':2,'2':3,'3':4}))
>>>[1, 2, 3]

重置索引列

df.reset_index(drop=True) #将索引index列重置，drop=true是避免将旧索引添加为列

利用loc、iloc提取行列数据

data.loc['a']   #取索引为a的行

data.loc[:,['a']]  #取‘a’列

data.loc[['a','b'],['A','B']] #取a,b行与A,B列的相交数

data.loc[:,:] #取所有数据

iloc基本语法类似 只是需指定第几行第几列
data.iloc[[0]] #取第一行

排序

1.Series排序

  • Series.sort_index 索引排序
  • Series.sort_values 值引排序

2.DataFrame排序

  • DataFrame.sort_index 索引排序
  • DataFrame.sort_values 值引排序
df = pd.DataFrame(np.arange(8).reshape((2, 4)),index = [3,2],columns = ['a','b','c','d'])
df.sort_index()                             #index按照列索引升序排列
df.sort_index(axis = 1)                     #让列索引升序排序
df.sort_index(axis = 1,ascending = False)   #让列索引降序排序
#让任选a,b两列数据同时降序排列
df.sort_values(by =['a','b'],ascending = [False,False])

数据清洗及特征处理

1.处理空缺值

1.1.dropna函数

  • 用来移除空缺值
DataFrame.dropna(axis=0, how='any', thresh=None, subset=None,inplace=False)
df.dropna()               		  #只保留行列都没有空值的
df.dropna(axis='columns') 		  #只保持某行中没有空值的
df.dropna(how='all')      		  #只去除所有元素缺失的
df.dropna(thresh=2)       		  #默认去除某行有两个缺失的，更改axis可以变成列
df.dropna(subset=['name', 'toy']) #指定两列去除缺失

1.2.fillna函数

  • 填充空缺值
DataFrame.fillna(value=None,method=None,axis=None,inplace=False,limit=None,downcast=None)
df.fillna(0) #用0填充空缺值
#A列用0填充
values = {"A": 0, "B": 1, "C": 2, "D": 3}
df.fillna(value=values)

2.处理重复值

df[df.duplicated()]       #查看重复数据
df = df.drop_duplicates() #清除重复数据

特征观察与处理

1.将连续变量进行离散化处理

#将连续变量Age平均分箱成5个年龄段，并分别用类别变量12345表示
df['AgeBand'] = pd.cut(df['Age'], 5,labels = [1,2,3,4,5])
#将连续变量Age划分为(0,5] (5,15] (15,30] (30,50] (50,80]五个年龄段，并分别用类别变量12345表示
df['AgeBand'] = pd.cut(df['Age'],[0,5,15,30,50,80],labels = [1,2,3,4,5])
#将连续变量Age按10% 30% 50 70% 90%五个年龄段，并用分类变量12345表示
df['AgeBand'] = pd.cut(df['Age'],[0,0.1,0.3,0.5,0.7,0.9],labels = [1,2,3,4,5])

2.对文本变量进行转换

#查看类别文本变量名及种类
df['Sex'].value_counts()
#将类别文本转换为12345,此处是把性别用1,2来量化
df['Sex_num'] = df['Sex'].replace(['male','female'],[1,2])
df['Sex_num'] = df['Sex'].map({'male': 1, 'female': 2})
